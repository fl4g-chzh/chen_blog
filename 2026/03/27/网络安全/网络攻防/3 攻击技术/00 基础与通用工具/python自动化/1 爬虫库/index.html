requestsеә“

requests жҳҜ Python з”ҹжҖҒдёӯжңҖдёәе№ҝжіӣдҪҝз”Ёзҡ„ HTTP е®ўжҲ·з«Ҝеә“гҖӮеңЁзҪ‘з»ңе®үе…ЁиҮӘеҠЁеҢ–еңәжҷҜпјҲеҰӮиө„дә§жҺўжөӢгҖҒзӣ®еҪ•зҲҶз ҙгҖҒжјҸжҙһ PoC зј–еҶҷпјүдёӯпјҢе®ғд»Ҙй«ҳеәҰе°ҒиЈ…зҡ„ API жҸҗдҫӣдәҶжһҒе…¶дҫҝжҚ·зҡ„зҪ‘з»ңдәӨдә’иғҪеҠӣгҖӮ

дёҖгҖҒ еҹәзЎҖйҖҡдҝЎжңәеҲ¶ (HTTP Methods)

еңЁзј–еҶҷиҮӘеҠЁеҢ–и„ҡжң¬ж—¶пјҢжңҖеёёз”Ёзҡ„йҖҡдҝЎж–№ејҸдёә GET дёҺ POST иҜ·жұӮгҖӮ

  • GET иҜ·жұӮ (дҝЎжҒҜиҺ·еҸ–/жҺўжөӢ): еёёз”ЁдәҺ URL еҸӮж•°дј йҖ’дёҺйЎөйқўжҠ“еҸ–гҖӮ

  • POST иҜ·жұӮ (ж•°жҚ®жҸҗдәӨ/дәӨдә’): еёёз”ЁдәҺиЎЁеҚ•жҸҗдәӨгҖҒJSON иҪҪиҚ·еҸ‘йҖҒжҲ–ж–Үд»¶дёҠдј жјҸжҙһжөӢиҜ•гҖӮ

дәҢгҖҒ ж ёеҝғе“Қеә”еҜ№иұЎи§Јжһҗ (Response Object)

еӨ„зҗҶзӣ®ж ҮжңҚеҠЎеҷЁиҝ”еӣһзҡ„ж•°жҚ®жҳҜзҲ¬иҷ«дёҺжү«жҸҸеҷЁзҡ„е…ій”®жӯҘйӘӨгҖӮд»ҘдёӢжҳҜеҝ…йЎ»жҺҢжҸЎзҡ„е“Қеә”еұһжҖ§пјҡ

еұһжҖ§/ж–№жі• иҝ”еӣһзұ»еһӢ е®һжҲҳеә”з”ЁеңәжҷҜ
r.status_code int зҠ¶жҖҒз Ғж ЎйӘҢпјҡеҲӨж–ӯйЎөйқўжҳҜеҗҰеӯҳеңЁ (200, 404) жҲ–жҳҜеҗҰеӯҳеңЁжқғйҷҗжҺ§еҲ¶ (401, 403)гҖӮ
r.text str ж–Үжң¬и§ЈжһҗпјҡжҸҗеҸ– HTML жәҗз ҒпјҢз”ЁдәҺжӯЈеҲҷеҢ№й…ҚжҲ– BeautifulSoup и§ЈжһҗжҸҗеҸ–ж•Ҹж„ҹдҝЎжҒҜгҖӮ
r.content bytes дәҢиҝӣеҲ¶жөҒиҺ·еҸ–пјҡдёӢиҪҪйӘҢиҜҒз ҒеӣҫзүҮгҖҒеҸҜжү§иЎҢж–Үд»¶жҲ–еӨ„зҗҶйқһ UTF-8 зј–з Ғзҡ„зү№ж®ҠйЎөйқўгҖӮ
r.headers dict жҢҮзә№иҜҶеҲ«пјҡиҜ»еҸ– Server жҲ– X-Powered-By еӯ—ж®өпјҢиҜҶеҲ«зӣ®ж Ү Web е®№еҷЁеҸҠзүҲжң¬гҖӮ
r.json() dict API дәӨдә’пјҡеҝ«йҖҹеҸҚеәҸеҲ—еҢ– RESTful API иҝ”еӣһзҡ„ JSON ж јејҸж•°жҚ®гҖӮ

дёүгҖҒ иҝӣйҳ¶й…ҚзҪ® (й’ҲеҜ№еҸҚзҲ¬дёҺе®үе…Ёи§„йҒҝжңәеҲ¶)

еңЁе®һйҷ…зҡ„зҪ‘з»ңж”»йҳІдёӯпјҢзӣ®ж Үзі»з»ҹйҖҡеёёйғЁзҪІдәҶ WAF (Web еә”з”ЁйҳІзҒ«еўҷ) жҲ–еҸҚзҲ¬зӯ–з•ҘгҖӮзҶҹз»ғиҝҗз”Ёд»ҘдёӢеҸӮж•°жҳҜз»•иҝҮеҹәзЎҖйҳІжҠӨзҡ„еүҚжҸҗпјҡ

  • 1. дјӘйҖ иҜ·жұӮеӨҙ (Headers Spoofing):

    й»ҳи®Өжғ…еҶөдёӢпјҢrequests зҡ„ User-Agent жҳҜ python-requests/x.x.xпјҢиҝҷдјҡиў«еӨ§еӨҡж•°е®үе…Ёи®ҫеӨҮзӣҙжҺҘжӢҰжҲӘгҖӮеҝ…йЎ»иҝӣиЎҢдјӘиЈ…гҖӮ

  • 2. д»ЈзҗҶжұ жҺҘе…Ҙ (Proxy Routing):

    дёәйҳІжӯўеҚ• IP жҺўжөӢйў‘зҺҮиҝҮй«ҳиў«е°ҒзҰҒпјҢйңҖз»“еҗҲд»ЈзҗҶжұ дҪҝз”ЁгҖӮ

  • 3. еҝҪз•Ҙ SSL иҜҒд№Ұж ЎйӘҢ (SSL Verification):

    еңЁжөӢиҜ•иҮӘзӯҫеҗҚиҜҒд№Ұзҡ„еҶ…зҪ‘зі»з»ҹжҲ–иў«дёӯй—ҙдәәд»ЈзҗҶжҠ“еҢ…ж—¶пјҢеёёдјҡйҒҮеҲ° SSL й”ҷиҜҜгҖӮ

  • 4. и¶…ж—¶жҺ§еҲ¶ (Timeout Control):

    жү«жҸҸжө·йҮҸиө„дә§ж—¶пјҢйҒҝе…Қеӣ зӣ®ж Үдё»жңәдёҚе“Қеә”иҖҢеҜјиҮҙи„ҡжң¬ж— йҷҗжңҹжҢӮиө·пјҲзәҝзЁӢйҳ»еЎһпјүгҖӮ

еӣӣгҖҒ дјҡиҜқз»ҙжҢҒ (requests.Session)

еңЁйңҖиҰҒиә«д»Ҫи®ӨиҜҒзҡ„жөӢиҜ•еңәжҷҜпјҲеҰӮеҗҺеҸ°жјҸжҙһеҲ©з”ЁпјүдёӯпјҢдҪҝз”Ё Session еҜ№иұЎеҸҜд»ҘиҮӘеҠЁеӨ„зҗҶе’Ңи·ЁиҜ·жұӮдҝқжҢҒ CookiesпјҢж— йңҖжҜҸж¬ЎжүӢеҠЁжҸҗеҸ–е’Ңйҷ„еҠ гҖӮ

BeautifulSoupеә“

BeautifulSoupпјҲз®Җз§° BS4пјүжҳҜдёҖдёӘиғҪеӨҹд»Һ HTML жҲ– XML ж–Үд»¶дёӯжҸҗеҸ–ж•°жҚ®зҡ„ Python еә“гҖӮе®ғйҖҡиҝҮиҪ¬жҚўж–ҮжЎЈж ‘пјҢдёәеӨҚжқӮзҡ„ DOM з»“жһ„жҸҗдҫӣдәҶз®ҖеҚ•гҖҒPythonic зҡ„еҜјиҲӘгҖҒжҗңзҙўе’Ңдҝ®ж”№жҺҘеҸЈгҖӮ

дёҖгҖҒ ж ёеҝғи§ЈжһҗеҷЁйҖүжӢ©дёҺеҲқе§ӢеҢ–

еңЁзҪ‘з»ңж”»йҳІе’ҢиҮӘеҠЁеҢ–жү«жҸҸдёӯпјҢзӣ®ж ҮзҪ‘йЎөзҡ„з»“жһ„еҫҖеҫҖе№¶дёҚи§„иҢғпјҲеӯҳеңЁж ҮзӯҫзјәеӨұгҖҒй—ӯеҗҲй”ҷиҜҜзӯүпјүгҖӮйҖүжӢ©еҗҲйҖӮзҡ„и§ЈжһҗеҷЁиҮіе…ійҮҚиҰҒпјҡ

и§ЈжһҗеҷЁ иҜӯжі• дјҳеҠҝ еҠЈеҠҝ жҺЁиҚҗеңәжҷҜ
html.parser BeautifulSoup(html, "html.parser") Python еҶ…зҪ®пјҢж— йңҖйўқеӨ–е®үиЈ…гҖӮ е®№й”ҷиғҪеҠӣдёҖиҲ¬пјҢйҖҹеәҰйҖӮдёӯгҖӮ иҪ»йҮҸзә§и„ҡжң¬пјҢзҺҜеўғеҸ—йҷҗж—¶дҪҝз”ЁгҖӮ
lxml BeautifulSoup(html, "lxml") йҖҹеәҰжһҒеҝ«пјҢе®№й”ҷиғҪеҠӣжһҒејәгҖӮ йңҖиҰҒе®үиЈ… C иҜӯиЁҖдҫқиө–еә“ (pip install lxml)гҖӮ е®үе…Ёжү«жҸҸгҖҒеӨ§и§„жЁЎзҲ¬иҷ«зҡ„йҰ–йҖүгҖӮ

ж ҮеҮҶеҲқе§ӢеҢ–д»Јз Ғпјҡ

дәҢгҖҒ ж ёеҝғжҗңзҙўдёҺе®ҡдҪҚж–№жі• (DOM йҒҚеҺҶ)

еңЁиҮӘеҠЁеҢ–и„ҡжң¬дёӯпјҢжңҖеёёз”Ёзҡ„ж“ҚдҪңжҳҜд»Һжө·йҮҸжәҗз ҒдёӯзІҫеҮҶжҸҗеҸ–зү№е®ҡж ҮзӯҫгҖӮ

  • 1. find() дёҺ find_all() (еҹәдәҺж ҮзӯҫдёҺеұһжҖ§зҡ„жҸҗеҸ–):

    • find(): иҝ”еӣһеҢ№й…ҚеҲ°зҡ„з¬¬дёҖдёӘе…ғзҙ пјҢеёёз”ЁдәҺе®ҡдҪҚе”ҜдёҖзҡ„зҷ»еҪ•иЎЁеҚ•жҲ–зү№е®ҡзҡ„ж ҮйўҳгҖӮ

    • find_all(): иҝ”еӣһеҢ…еҗ«жүҖжңүеҢ№й…Қе…ғзҙ зҡ„еҲ—иЎЁпјҢеёёз”ЁдәҺйҒҚеҺҶжүҖжңүй“ҫжҺҘжҲ–еӣҫзүҮгҖӮ

  • 2. CSS йҖүжӢ©еҷЁ select() (жӣҙзҒөжҙ»зҡ„е®ҡдҪҚж–№ејҸ):

    ж”ҜжҢҒзұ»дјј jQuery зҡ„ CSS йҖүжӢ©еҷЁиҜӯжі•пјҢеңЁеӨ„зҗҶеұӮзә§жһҒж·ұзҡ„еӨҚжқӮйЎөйқўж—¶жҜ” find_all жӣҙз®ҖжҙҒгҖӮ

дёүгҖҒ зҪ‘з»ңж”»йҳІе®һжҲҳеңәжҷҜеә”з”Ё

еңЁзј–еҶҷе®үе…Ёе·Ҙе…·ж—¶пјҢBS4 зҡ„еә”з”ЁеңәжҷҜйҖҡеёёйқһеёёжҳҺзЎ®гҖӮд»ҘдёӢжҳҜдёүдёӘй«ҳйў‘е®һжҲҳд»Јз ҒжЁЎжқҝпјҡ

еңәжҷҜ 1пјҡжҸҗеҸ–йЎөйқўжүҖжңүй“ҫжҺҘ (з”ЁдәҺзҲ¬иҷ«зӣ®еҪ•йҒҚеҺҶдёҺж”»еҮ»йқўжү©еұ•)

жҲ‘д»¬йңҖиҰҒжҸҗеҸ– <a> ж Үзӯҫзҡ„ href еұһжҖ§пјҢд»Ҙж”¶йӣҶзӣ®ж Үз«ҷзӮ№зҡ„и·Ҝз”ұдҝЎжҒҜгҖӮ

еңәжҷҜ 2пјҡиҺ·еҸ–иЎЁеҚ•йҡҗи—Ҹеӯ—ж®ө (иҮӘеҠЁеҢ–еҲ©з”ЁгҖҒз»•иҝҮ Anti-CSRF Token)

еңЁзј–еҶҷжјҸжҙһеҲ©з”Ё PoC ж—¶пјҢз»ҸеёёйңҖиҰҒе…Ҳ GET иҜ·жұӮйЎөйқўпјҢжҸҗеҸ–йҡҗи—Ҹзҡ„ tokenпјҢеҶҚеёҰе…Ҙ POST иҜ·жұӮдёӯгҖӮ

еңәжҷҜ 3пјҡжҸҗеҸ– HTML жіЁйҮҠ (ж•Ҹж„ҹдҝЎжҒҜжі„йңІжҺўжөӢ)
